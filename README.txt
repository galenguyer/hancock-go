NAME:
   hancock - a simple certificate manager

USAGE:
   hancock [global options] command [command options] [arguments...]

COMMANDS:
   init                initialize the certificate authority
   new, create, issue  sign a new key for a host
   renew               renew expiring keys
   help, h             Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help (default: false)
